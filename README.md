# SPACE CHAT

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## What it does

**Space chat** : your space chat bot 

Space chat is a chatbot that can help you with your space research !! 
Space chat is not only a chatbot, but a true friend to get to know and enjoy.


Thanks to the search bar and **star road**, space chat is able to make the user see the constellations used. 

## Future 
In the future, the space map could be expanded, so as not only to show the constellations, but a simple animation of what is explained.
